## What changed

v2.0.0 is a major release. The biggest change: **ETH and SOL have been removed.**

We made the decision to focus 100% of our infrastructure on **Bitcoin (BTC)** and **Tether (USDT)** only.

Why:
- ETH & SOL represented a very small share of actual usage
- They were consuming a large part of our server resources
- Removing them allowed us to dedicate everything to BTC & USDT
- Result: faster broadcast, lower latency, better relay stability

## New in v2.0.0

**🌐 Integrated VPN**
Full VPN system built directly inside the software. WireGuard / OpenVPN / IKEv2. Kill Switch. DNS leak protection. 15+ server locations. No third-party app required.

**✖ Cancel Broadcast**
Ability to cancel any transaction at any point during the broadcast sequence. Confirmation modal. Instant engine reset. No funds sent, no trace left.

**🔔 Windows Notifications**
Native toast notifications for broadcast, TXID confirmed and wallet loaded events — even when the software is minimized.

**⚡ Auto Fee Oracle**
Real-time fee fetched from Blockstream mempool on every activation.

**🎨 Full UI Redesign**
Complete interface overhaul — new layout, new panels, improved transaction grid, new log system.
